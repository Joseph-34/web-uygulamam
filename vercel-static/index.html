Bu site Vercel’de sadece bilgi sayfası

Uygulama ASP.NET Core (MVC, SignalR, SQL Server). Vercel bu sunucu türünü çalıştırmaz; bu yüzden tam sürüm burada yok.

Çalışan sürüm için repodaki Dockerfile ile Railway, Render veya Azure App Service kullanın; veritabanı bağlantısını ortam değişkeniyle verin.
